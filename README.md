# NoctisOutlive

AIに要件のみ渡して作成してもらったらどうなるかの検証

```
ヴァンパイアサバイバーのようなゲームを作成していきます。

以下要件です。
* node.js(v22) typescript (webgl使用)
* ヴァンパイアサバイバーのようなサバイバーゲームを作成
* アイテムの保持を決まった用量の2dバックパックを整理して保持する方式とし、アイテムの中にはバックパック拡張も含めます。
* 宝箱(ルートボックス)は3択(アイテムで増減など)からアイテムやバックパックのマス目の部品などを1つ選ぶ方式とします。
* 敵を倒すと経験値をその場に落としプレイヤーが拾う必要がある標準方式です。
* 1ステージごとに中ボスと大ボスがいるものとします。
* 1ページでローグライクのステージ選択、プレイするステージ、その他色々を全てを1画面に納めてください。(フラグで管理など最も最適な方法を使用して下さい。)
* ステージは、標準ステージ(敵と戦う)、高難易度ステージ(強敵と戦う)、ショップ(アイテムなどを購入、売却する)、イベントステージ(アイテムがもらえたり、強化、損失など少しの会話と選択肢で結果が変換)
* 敵の画像やアイテム画像をメモリを圧迫させず、尚且つ無駄のない方法で管理して下さい。
* 私は絵を描く事が出来ないため、画像は静止画を主として、エフェクトなどで動きを誤魔化してください。
* 効果音を再生する機構も作成して下さい。
* ファイルを適切に分割して、管理性を向上する。
* webassembly(rust)を使用可能な場合は使用し、実行速度の高速化と安定化を実施して下さい。
* WebWorkerを使用可能な場合は使用し、メインスレッドのブロックを最小限に抑えてください。
* ヴァンパイアサバイバーがわからない場合は調べて理解を深めてから作成を開始して下さい。
* コード全文を複数の回答にまたがっても良いので最も省略なく、妥協なしのコード全文を作成して下さい。
* tsのビルドはbuild.jsを使用してビルドし、esbuildなど最新で、高速、安定性の高いビルド方法を複数組合せ、実装して下さい。
* バグ修正を実施してください。
* 作成後に最後にビルドして検証して下さい。
上記要件から現在の実装に足りない部分を妥協なしで作成して下さい。
既存のコードに作成を妨げる要素が存在する場合は、改善しても構いません。
```
